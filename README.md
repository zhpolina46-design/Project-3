# Проект III. Объекты культурного наследия г. Калининграда

**Карта онлайн:** https://zhpolina46-design.github.io/Project-3/

## О проекте

На карте показаны объекты культурного наследия федерального, регионального, местного значения города Калининграда и их анализ. Основной точечный слой ОКН представлен кластеризацией с градацией маркера по статусу объекта. При наведении курсора на объект при нажатии возможно увидеть более точную информацию по памятнику или достопримечательному месту. Также на основе этих данных созданы новые аналитические слои распределения ОКН по районам города (этот слой сделан при помощи Chloroplet). Также с помощью heatmap можнон увидеть тепловую карту концентрации ОКН, построенных во времена Кенигсберга (до 1945 г.). Еще один тематический слой - землепользование по категориям. Можно включать и выключать все эти слои и переходить в полноэкранный режим.

## Что изображено на карте

Количество ОКН по районам: 
В Центральном районе города сосредоточено наибольшее число ОКН - 207. В Ленинградском районе - 159, а в Московском - 62. 

Количество ОКН по слоям землепользования:
В основном все объекты культурного наследия расположены в жилых зонах в количестве 248. В промышленной зоне - 8, в коммерческой - 6, в зеленых зонах по одному ОКН. 

Количество ОКН, построенных во времена Кенигсберга:
На тепловой карте показана концентрация 378 объектов культурного наследия, построенных до 1945 из 433 текущих города Калининграда.
